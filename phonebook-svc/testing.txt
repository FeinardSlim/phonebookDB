Untuk melakukan ujicoba dan debugging dari pycharm, gunakan port 32006

karena menggunakan bridged mode , untuk mengakses ke phonebookdb-service,
harus menggunakan ip address bridge tersebut, untuk mengetahui IPnya, gunakan
docker inspect phonebookdb-service, carilah baris yang berisikan "IPAddress"

untuk melihat daftar phonebook
 curl -X GET http://[IPADDRESS]:32000/phones

untuk membuat record di phonebook
  curl -X POST http://[IPADDRESS]:32000/phones  \
  -d '{"nama": "Edinson Cavani", "alamat" : "Uruguay", "notelp": "781723"}' \
  -H "Content-Type: application/json"

untuk mendapatkan detail record dengan berbasis id
 curl -X GET http://[IPADDRESS]:32000/phones/fe437958-2fa4-11eb-bf35-7fc0bd24c845

untuk menghapus record dengan berbasis id
 curl -X DELETE http://[IPADDRESS]:32000/phones/fe437958-2fa4-11eb-bf35-7fc0bd24c845

unutk mengupdate record dengan berbasis id
  curl -X PUT http://[IPADDRESS]:32000/phones/fe437958-2fa4-11eb-bf35-7fc0bd24c845 \
  -H 'Content-type: application/json' \
  -d '{"nama": "Bruno Fernandes"}'
